## üöÄ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/adityadomle/BizFlow.git
   cd BizFlow
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Run the development server**
   ```bash
   npm run dev
   ```
4. **Open [localhost:5173](http://localhost:5173) in your browser**
